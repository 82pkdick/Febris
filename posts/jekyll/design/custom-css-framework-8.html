カラーパレットを作ってみよう

前に作った Card と Grid Layout を使った仕組みで、１４０色+α のカラーパレットを作ってみよう。

こういう細かいものを作っていくと部分部分で paddingとかfont-sizeとかいじりたくなるが、Jekyll の markdown は HTML をまんま処理してくれるので、ある記事の中でしか使わないようなスタイル設定は、記事先頭の Front-Matter の外、すぐ下に<style>〜</stayle>で書いておくという方法もある。

ここでは次のような CSS クラス.color-cardを記事先頭に設定して、各カードに割り当てた。.bg-xxxxxxが背景色設定、カードを囲む全体のグリッドブロックに付けた.grid-gap0クラスは、グリッド間隔を０にする設定。これらは自作フレームワークに新しく作った。

<style>
.color-card {
  text-align: center;
  padding-top: 20px;  /* 文字天地中央にする(ゴマカシｗ) */
  font-size: 0.9em;
}
</style>

.bg-light-gray1 { background: $light-gray1; } // <--色名部分を変えていくだけ

.grid-gap0 {
  grid-gap: 0;
}

[markdown]

> light-gray1
> {: .card .bg-light-gray1 .color-card}
> light-gray2
> {: .card .bg-light-gray2 .color-card}
> light-gray3
> {: .card .bg-light-gray3 .color-card}
> light-gray4
> {: .card .bg-light-gray4 .color-card}
> light-gray5
> {: .card .bg-light-gray5 .color-card}
> mid-gray1
> {: .card .bg-light-gray6 .color-card}
  ......
> dark-gray3
> {: .card .bg-light-gray1 .color-card}
> dark-gray4
> {: .card .bg-light-gray1 .color-card}
{: .card .grid-col5 .grid-gap0 .border-none}

全ての色を定義したファイルを、Jekyllのサイトルート直下の_sass/framework-modules/_color-palette.scssとした。ついでに140色すべての文字色指定.fc-xxxと背景色指定bg-xxxを作った。手間がかかるように思えるが、Vscode の変態マルチカーソル使えばあっという間です（楽w）。
