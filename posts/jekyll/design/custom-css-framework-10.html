Border

Table で markdown から スタイルをアレンジするやり方が分かったので、ボーダー値の変更に応用してみる。 この設定は_block-style.scssに追加すればいいと思う。

[_block-style.scss に追記]

// -------- border style -------------

.border {
  // CSS Custom Variables Default
  --bdr-set: 1px solid #cdcdcd;
  --bdr-width: 1px;
  --bdr-style: solid;
  --bdr-color: #cdcdcd;
  --bdr-lwd: 1px;
  --bdr-rwd: 1px;
  --bdr-twd: 1px;
  --bdr-bwd: 1px;
  --bdr-rad: 2px;
}

.border { border: var(--bdr-set); }

.border {
  &.bdr-width { border-width: var(--bdr-width); }
  &.bdr-style { border-style: var(--bdr-style); }
  &.bdr-color { border-color: var(--bdr-color); }
  &.bdr-lwd { border-left-width: var(--bdr-lwd); }
  &.bdr-rwd { border-right-width: var(--bdr-rwd); }
  &.bdr-twd { border-top-width: var(--bdr-twd); }
  &.bdr-bwd { border-bottom-width: var(--bdr-bwd); }
  &.bdr-rad { border-radius: var(--bdr-rad); }
}

CSS変数の使い方は Table で説明したので詳細は省く。

.border クラス

単に.borderクラスだけを指定するとデフォルト値のボーダーが付く。他に付加しているクラスは他の関連記事参照と言いたいが、説明していないのもありーの、めんどくさいーので、名前から想像して下さいメンゴ。

[markdown]

Box with Border
{: .border .pad-10 .wd-30 .align-c}

Box with Border

–bdr-set CSS変数

さらに--bdr-setCSS変数で一括指定

[markdown]

Box with Border Set
{: .border .pad-10x20 .wd-30 .align-c}
{: style="--bdr-set:8px double lime;"}

Box with Border Set

.bdr-width

.bdr-widthボーダーの太さ、.bdr-styleスタイル、.border-color色、それぞれ個別に指定するクラス。ただしこの3種でボーダーを表示するためにはデフォルトクラス.borderが必要。

.bdr-widthボーダーの太さ。--bdr-widthで太さを指定する。数値を省略した場合はデフォルトの 1px

[markdown]

Box with 5px Border
{: .border .bdr-width .pad-10x20 .wd-30 .align-c}
{: style="--bdr-width: 5px;"}

Box with 5px Border

.bdr-style

.bdr-styleボーダーのスタイル。省略時のデフォルトは solid

[markdown]

Box with Dashed Border
{: .border .bdr-style .bdr-width .pad-10x20 .wd-40 .align-c}
{: style="--bdr-style:dashed; --bdr-width: 3px"}

Box with Dashed Border

.bdr-color

.bdr-colorボーダーの色。省略時のデフォルトは明るいグレイ（#cdcdcd）

[markdown]

Box with color Border
{: .border .bdr-color .pad-10x20 .wd-40 .align-c}
{: style="--bdr-color:red;"}

Box with color Border

組合せ例

ボーダーの色がピンクで、スタイルがdashedの太いボーダー。

Box with Pink Dashed Border

上の.bdr-styleを省略してみたら、デフォルトのsolidになった。

Box with Pink Solid Border

色も省略したもの。デフォルトのグレーになる

Box with Default Gray Solid Border, Width:10px

各辺の太さ

左辺の太さ: .bdr-lwd
右辺の太さ: .bdr-rwd
上辺の太さ: .bdr-twd
下辺の太さ: .bdr-bwd

左右だけ10px, 上下は2px

[markdown]

Box with Border
{: .border .bdr-width .bdr-color .bdr-lwd .bdr-rwd .pad-10x20 .wd-40 .align-c}
{: style="--bdr-width:2px; --bdr-color:gold; --bdr-lwd:10px; --bdr-rwd:10px;"}

Box with Border

左右は3px, 上は15px, 下は10px

[markdown]

Box with Border
{: .border .bdr-width .bdr-color .bdr-twd .bdr-bwd .pad-10x20 .wd-40 .align-c}
{: style="--bdr-width:3px; --bdr-color:olivedrab; --bdr-twd:15px; --bdr-bwd:10px;"}

Box with Border

要らない辺の数値を0にすれば、そこのボーダーを消すことになります。下辺だけ残した例。

Box with Bottom Border

角丸のボーダー

.bdr-radで角丸、style="--bdr-rad:数値"で値指定。

[markdown]

Box with Border Radius
{: .border .bdr-width .bdr-color .bdr-rad .pad-10x20 .wd-40 .align-c}
{: style="--bdr-width:3px; --bdr-color:royalblue; --bdr-rad:10px;"}

Box with Border Radius
